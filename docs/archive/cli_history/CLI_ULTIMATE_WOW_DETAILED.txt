import { TrendingUp, TrendingDown, Activity } from 'lucide-react';

                <button
                  key={tab}

                      ? 'bg-blue-500/20 text-blue-400 border border-blue-500/30'

            </div>
          </header>

                <motion.div
                  initial={{ opacity: 0, y: 20 }}

                  {/* Overview content */}
                </motion.div>
              )}
              {/* Other tabs */}
            </AnimatePresence>
          </div>
        </div>
      </main>
    </div>

## UNIVERSAL REQUIREMENTS (All 38 Pages)

### Code Quality Standards:
1. **TypeScript**: Strict typing, no any
2. **Imports**: Organized (React → Libraries → Components → Utils)
3. **Components**: Composition over monoliths (<300 lines)
4. **Naming**: Descriptive, consistent (camelCase vars, PascalCase components)
5. **Comments**: Business logic only, not obvious code

### UI/UX Standards:
1. **Spacing**: 
   - Container padding: p-6
   - Card spacing: space-y-6
   - Element gaps: gap-4
2. **Colors**:
   - Success: emerald-400/500
   - Danger: red-400/500
   - Info: blue-400/500
   - Warning: yellow-400/500
3. **Typography**:
   - Headers: text-2xl font-bold
   - Subheaders: text-sm text-zinc-400
   - Body: text-base
4. **Animations**:
   - Entrance: fade up 20px, duration 0.3s
   - Hover: scale 1.02, duration 0.2s
   - Stagger: 0.1s delay between children

### Responsive Breakpoints:
- Mobile: < 768px (stack, single column)
- Tablet: 768px - 1024px (2 columns)
- Desktop: > 1024px (full layout)

### Performance:
- Lazy load charts (React.lazy)
- Debounce search (300ms)
- Virtual scroll for >100 items
- Code split by route

### Accessibility:
- ARIA labels on interactive elements
- Keyboard navigation support
- Focus indicators visible
- Color contrast WCAG AA
