# Agent Deployment Plan

**Date**: 2025-11-27
**Strategy**: Just-in-Time (JIT) Hiring. Deploy agents only when needed to maximize cash flow.

## 🛠️ Phase 1: The Builders (Weeks 1-4)

### 1. DevAgent (Day 1)
**Role**: Code Generation & Maintenance.
**Why Now**: Need to build the AI Core immediately.
**Setup**:
- Connect Cursor AI to Repo.
- Set Claude 3.5 Sonnet as reviewer.
**Cost**: $100/mo.

### 2. OpsAgent (Week 1)
**Role**: Monitoring & Stability.
**Why Now**: To catch errors during the "Brain Build".
**Setup**:
- Connect Sentry & Supabase logs.
**Cost**: $50/mo.

### 3. DataAgent (Week 2)
**Role**: Model Training.
**Why Now**: AI features need data training before launch.
**Setup**:
- Connect to Binance Historical API.
- Setup PyTorch environment.
**Cost**: $100/mo.
