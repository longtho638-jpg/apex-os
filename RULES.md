# Project Rules

## 1. Domain Consistency
- **Canonical Domain:** `https://apexrebate.com`
- **Incorrect Domains:** `https://apex-os.io`
- **Usage:** Always use the canonical domain in links, metadata, and API responses.

## 2. Terminology
- **Referral System:** Use "Referral" for the feature name, but "Affiliate" for the dashboard path (`/dashboard/affiliate`) is acceptable if legacy.
- **Viral Economics:** Refers to the underlying engine/schema.

## 3. Code Style
- **Components:** Use `src/components` directory.
- **Icons:** Use `lucide-react`.
